# Требования к приложению ПиццаДогстер

## Функциональные требования
- Просмотр меню с пиццами и хот-догами
- Добавление товаров в корзину
- Оформление заказа с указанием адреса
- Онлайн-оплата картой
- Отслеживание статуса доставки

## Нефункциональные требования
- Приложение должно работать быстро (ответ < 2 секунд)
- Данные заказов хранятся в БД
- Интерфейс простой и понятный
